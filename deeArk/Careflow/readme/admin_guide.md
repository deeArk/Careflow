# Admin Guide

CareFlow administrators manage access, users, and system oversight.

Use this guide for day-to-day admin tasks.

Use the [Support center](../support/help_center.md) for common issues and quick answers.

Use the [Staff User Guide](staff-user-guide.md) for day-to-day patient workflows.

## User Management

<details>

<summary>Create a staff user</summary>

{% stepper %}
{% step %}
## Open the **Staff Search and Retrieval Screen**
{% endstep %}

{% step %}
## Click **Register New Staff**
{% endstep %}

{% step %}
## Select the staff role

* Register Doctor
* Register Nurse
* Register Admin
* Cashier
{% endstep %}

{% step %}
## Complete **Personal Information**
{% endstep %}

{% step %}
## Complete **Employment Information**
{% endstep %}

{% step %}
## Complete **Account Setup**
{% endstep %}

{% step %}
## Review the details in **Review & Confirm**
{% endstep %}

{% step %}
## Submit the registration

The success screen confirms the new staff account.
{% endstep %}
{% endstepper %}

</details>

<details>

<summary>Remove a staff user</summary>

{% stepper %}
{% step %}
## Open the **Staff Search and Retrieval Screen**
{% endstep %}

{% step %}
## Search for the staff user
{% endstep %}

{% step %}
## Open the user profile
{% endstep %}

{% step %}
## Click **Remove User**
{% endstep %}

{% step %}
## Confirm the removal

The user can no longer sign in after removal.
{% endstep %}
{% endstepper %}

</details>

<details>

<summary>Restore a removed staff user</summary>

{% stepper %}
{% step %}
## Open the **Staff Search and Retrieval Screen**
{% endstep %}

{% step %}
## Search for the removed staff user
{% endstep %}

{% step %}
## Open the user profile
{% endstep %}

{% step %}
## Click **Restore User**
{% endstep %}

{% step %}
## Confirm the restoration

The user can sign in again after the account is restored.
{% endstep %}
{% endstepper %}

</details>
