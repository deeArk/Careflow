# Admin Guide

CareFlow administrators manage access, users, and system oversight.

Use this guide for day-to-day admin tasks.

***

<details>

<summary>Cannot log in</summary>

* Check your username and password.
* Confirm your internet connection.
* Confirm your account has not been removed by another administrator.
* Contact another administrator if access still fails.

## User Management

<details>

<summary>Create a staff user</summary>

{% stepper %}
{% step %}
## Open **Settings**
{% endstep %}

{% step %}
## Open **User Management**
{% endstep %}

{% step %}
## Click **Add User**
{% endstep %}

{% step %}
## Enter staff details
{% endstep %}

{% step %}
## Assign a role

* Doctor
* Nurse
* Receptionist
* Admin
{% endstep %}

{% step %}
## Click **Save**

The new staff user can sign in after the account is created.
{% endstep %}
{% endstepper %}

</details>

<details>

<summary>Remove a staff user</summary>

{% stepper %}
{% step %}
## Open **Settings**
{% endstep %}

{% step %}
## Open **User Management**
{% endstep %}

{% step %}
## Search for the staff user
{% endstep %}

{% step %}
## Open the user profile
{% endstep %}

{% step %}
## Click **Remove User**
{% endstep %}

{% step %}
## Confirm the removal

The user can no longer sign in after removal.
{% endstep %}
{% endstepper %}

</details>

<details>

<summary>Restore a removed staff user</summary>

{% stepper %}
{% step %}
## Open **Settings**
{% endstep %}

{% step %}
## Open **User Management**
{% endstep %}

{% step %}
## Search for the removed staff user
{% endstep %}

{% step %}
## Open the user profile
{% endstep %}

{% step %}
## Click **Restore User**
{% endstep %}

{% step %}
## Confirm the restoration

The user can sign in again after the account is restored.
{% endstep %}
{% endstepper %}

</details>
