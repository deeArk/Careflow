# Admin Guide

CareFlow administrators manage access, users, and system oversight.

Use this guide for day-to-day admin tasks.

Use the [Support center](../support/help_center.md) for common issues and quick answers.

Use the [Staff User Guide](staff-user-guide.md) for day-to-day patient workflows.

Administrators are responsible for:

* Monitoring daily activity from **Dashboard**
* Managing staff accounts and access
* Reviewing patients, appointments, and departments
* Monitoring billing, payments, and reports
* Updating system settings

## User Management

<details>

<summary>Create a staff user</summary>

{% stepper %}
{% step %}
## Open the **Staff Search and Retrieval Screen**
{% endstep %}

{% step %}
## Click **Register New Staff**
{% endstep %}

{% step %}
## Select the staff role

* Doctor
* Nurse
* Administrator
* Cashier
{% endstep %}

{% step %}
## Complete **Personal Information**

This is step 1 of 4.

Enter:

* Full Name
* Date of Birth
* Phone Number
* Email
* Address

Click **Next** to continue.
{% endstep %}

{% step %}
## Complete **Employment Information**

This is step 2 of 4.

Enter:

* **Department**
* **Specialization**
* **License Number**
* **Years of Experience**
* **Employment Type**

**Employee ID** appears automatically after saving.

Click **Next** to continue.
{% endstep %}

{% step %}
## Complete **Account Setup**

This is step 3 of 4.

Enter:

* **Username**
* **Email**
* **Password**
* **Confirm Password**
* **Role**

Select the permissions required for the account:

* **View patient records**
* **Manage appointments**
* **Billing access**
* **Clinical history**
* **Manage staff**
* **System settings**

Click **Next** to continue.
{% endstep %}

{% step %}
## Review the details in **Review & Confirm**

This is step 4 of 4.

Review:

* **Personal Information**
* **Employment Information**
* **Account Information**

Use **Edit** if any section needs correction.

Click **Create Account** to finish.
{% endstep %}

{% step %}
## Submit the registration

The success screen confirms the new staff account.

The screen shows:

* **Employee ID**
* Temporary password
* Confirmation that login credentials were sent by email

You can then select **View Profile** or **Register Another**.
{% endstep %}
{% endstepper %}

</details>
