* Use [Quickstart](quickstart.md) for the fastest local setup.
* Use [Run Careflow locally](run-careflow-locally.md) if you need setup detail and verification steps.

## What you'll need

Before you start, make sure you have:

* [x] Access to the Careflow repository
* [x] Node.js and npm installed
* [x] A database connection string and JWT secret
* [x] A few minutes to run both backend and frontend
